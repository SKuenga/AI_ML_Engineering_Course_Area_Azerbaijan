# AI_ML_Engineering_Course_Area_Azerbaijan
A simple project based learning approach for students to actually learn and pursue a career in AI and ML Engineering.


# First Project:
Develop a simple CLI-Based Expense Tracking, Habit-Tracking or Task-Manager covering the basics of control flow, conditional statement, datastructure and functions. This particular project checks the basics of programming that are required as more abstract concepts and layers are added in learning.
